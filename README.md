**oer-science-labs** is an open library of **no-/low-cost, hands-on science experiment guides** for K–12 and early-college learners, each **aligned to curriculum standards** and **safety-reviewed for physical hazards** before it ships. The goal is concrete: let any teacher, tutor, homeschool parent, or after-school educator — including those with **no laboratory, no budget, and limited connectivity** — run a real, standards-aligned science investigation using household or inexpensive materials, without guessing at the safety risks.
